## commit 규칙
<code>commit convention</code>
* Feat: 새로운 기능 추가
* Design: UI 디자인 변경
* Fix: 버그 수정
* Docs: 문서 수정
* Style: 코드 포맷팅
* Refactor: 코드 리팩토링
* Gradle: gradle 파일 수정

<code>commit message</code>
* 태그: 메시지
* 예시
  * Design: BasicCompose UI 제작
